## 🎯 The dashboard (interactive)

The dashboard is a separate static HTML app. **It's not a Jekyll page** — it's served as-is.

| What | URL |
|---|---|
| Dashboard home | [/dashboard/](./dashboard/) |
| Dashboard README | [/dashboard/README.html](./dashboard/README.html) |

The dashboard has six modes (Revise / Quiz / Flashcards / Random Mix / Flashcards-All / Mastery Map). All state (confidence ratings) is persisted to your browser's `localStorage`.
